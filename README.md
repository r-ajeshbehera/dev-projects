🚀 Getting Started

Prerequisites

* Web server (XAMPP / Apache / Nginx)
* PHP or Node.js (depending on project)
* MySQL
* Git

Basic Setup Steps

1. Clone the repository:

   ```
   git clone https://github.com/r-ajeshbehera/dev-projects.git
   ```

2. Navigate to the desired project folder.

3. Set up the database (if applicable):

   ```
   mysql -u username -p database_name < setup.sql
   ```

4. Install dependencies (if required):

   ```
   composer install
   npm install
   ```

5. Run the project using your local server.

Detailed setup instructions are available inside each project folder.
